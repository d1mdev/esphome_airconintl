The "ASK STATUS" query packet is always F4F500400C00000101FE0100006600000001B3F4FB

A standard response packet is always like this:
F4F50140490100FE010101010066000104022A191A0F80808001910000000000000000C000800500001418000F3418000000000000000000000000000B0000000000000000000000000000000000064FF4FB

It is composed by:
F4F5				header
01					packet type (01 response)
40					padding byte
49					packet length (73 bytes plus 9 static F4F5014049..064FF4FB)
0100FE0101010100	padding bytes
6600				packet type and sub-tye (102 sub 0)
01					padding byte
[...]				data
064F				sum bit to bit of previous bytes, header excluded 
F4FB				footer

To read data we need to convert data from hex
04022A191A0F80808001910000000000000000C000800500001418000F3418000000000000000000000000000B0000000000000000000000000000000000

to bin
